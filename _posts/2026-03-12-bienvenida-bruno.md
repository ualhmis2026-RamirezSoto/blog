# Introducción al Proyecto
**Fecha**: 12 de marzo de 2026
**Autor**: Bruno

## Descripción del Proyecto
En este proyecto, nuestro equipo está desarrollando un blog estático utilizando Jekyll y desplegándolo en GitHub Pages y Azure. El objetivo es aprender sobre el desarrollo colaborativo, el uso de control de versiones con Git, y el despliegue de aplicaciones web.

Personalmente, me interesa mucho ver cómo todo esto se relaciona con la cultura DevOps, la automatización y el despliegue continuo.

## Herramientas Utilizadas
- Jekyll
- GitHub
- Visual Studio Code

¡Estamos emocionados por comenzar este proyecto y aprender juntos!
